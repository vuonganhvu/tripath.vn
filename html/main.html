Description

M090 is a Gateway for customer care, self-create package, buy packages, manage packages, top up balance, check data volume, check history, check news from phone, smartphone, computer or tablet. Free data while access via Mobifone network.

Features

  • Auto log in while on Mobifone 3G/4G network.
  • Check info for pre-paid / post-paid accounts.
  • Buy Mobifone packages.
  • Buy self-made packages.
  • Check packages info.
  • Buy more data for current packages / Cancel packages.
  • Check History.
  • Make payments for pre-paid, top up for post paid by any type of scratch card.
  • Check volume for current data plan.
  • Check bought packages.
  • Update personal info.
  • FAQ
  • Map of Mobifone stores
  • News
  • Connect with call center 9090
